# Strawberry Benchmark — Pangram Sentences (Spanish)
# Format: sentence|is_pangram|missing_letters
# Pipe-delimited (sentences may contain commas).
# Lines starting with # are comments. Blank lines are ignored.
#
# is_pangram: true if every letter a-z appears at least once
# missing_letters: comma-separated letters absent from near-pangrams (empty for true pangrams)
# Note: Spanish has ñ but for pangram purposes we check the 26 standard a-z letters.
# Accented vowels (á,é,í,ó,ú,ü) count as their base letter.

Aquel juglar explicaba muy bien que zafiros hechos valen poco y deben kitar del whisky.|true|
