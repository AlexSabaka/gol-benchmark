🎯 KEY FINDINGS (Quick Summary)
────────────────────────────────────────────────────────────────────────────────────

Finding #1: SYSTEM PROMPTS ARE REASONING SWITCHES
┌─────────────────────────────────────────────────────────────────────────┐
│ Same model, same task, different system prompt = different reasoning    │
│                                                                         │
│ Example: gemma3:1b on minimal prompts                                   │
│   Adversarial system → 27.78% accuracy                                 │
│   Analytical system  → 57.41% accuracy                                 │
│   Difference: +29.63 points (110% improvement!)                        │
└─────────────────────────────────────────────────────────────────────────┘

Finding #2: MODELS HAVE OPPOSITE PERSONALITIES
┌─────────────────────────────────────────────────────────────────────────┐
│ qwen3:0.6b LOVES adversarial (+18.5 points vs casual baseline)         │
│ gemma3:1b LOVES analytical (+22.4 points vs casual baseline)           │
│                                                                         │
│ This reflects their design: efficiency vs. quality                     │
└─────────────────────────────────────────────────────────────────────────┘

Finding #3: INSTRUCTION SATURATION EFFECT EXISTS
┌─────────────────────────────────────────────────────────────────────────┐
│ Too much guidance (both system+user) can HURT performance:             │
│                                                                         │
│ gemma3:1b with linguistic user prompts:                                │
│   Casual system:     83.33% (BEST)                                    │
│   Analytical system: 75.00% (DROPS 8.3 points - instruction clash!)  │
└─────────────────────────────────────────────────────────────────────────┘

Finding #4: USER PROMPT MATTERS MORE THAN SYSTEM PROMPT
┌─────────────────────────────────────────────────────────────────────────┐
│ Improvement from minimal → linguistic user prompts:                    │
│   qwen:  +28.7 points                                                  │
│   gemma: +48.3 points                                                  │
│                                                                         │
│ Investment in user prompt quality has massive ROI                     │
└─────────────────────────────────────────────────────────────────────────┘

Finding #5: MODEL ROBUSTNESS DIFFERS DRAMATICALLY
┌─────────────────────────────────────────────────────────────────────────┐
│ qwen3:0.6b: 35-63% range (28 points) = ROBUST to poor prompts         │
│ gemma3:1b:  27-83% range (56 points) = SENSITIVE to prompt engineering│
│                                                                         │
│ gemma performance is 1.93× MORE variable                               │
└─────────────────────────────────────────────────────────────────────────┘
